canvas

来源： Canvas的基本用法

canvas元素说明

<canvas id="tutorial" width="150" height="150"></canvas>

canvas标签只有width和height两个属性。该标签可以通过CSS来定义大小和其它样式，但在绘制图像时会伸缩以适应它的框架尺寸，因此建议在canvas的属性时规定宽高，而不是使用CSS

canvas默认宽高分别为150px和300px

canvas的结束标签不可省略

替换内容

较老的浏览器（尤其是IE9之前的浏览器）不支持canvas元素，因此需要展示替代内容

支持canvas的浏览器会忽略在容器中的内容，只会正常渲染canvas

<canvas id="stockGraph" width="150" height="150">
    current stock price: $3.15 + 0.15
</canvas>

<canvas id="clock" width="150" height="150">
    <img src="images/clock.png" width="150" height="150" />
</canvas>

渲染上下文(rendering context)

getContext()方法是canvas元素获得上下文和绘画功能，该方法只有一个参数，即上下文格式，对于2D图像：

var canvas = document.getElementById('tutorial');
var ctx = canvas.getContext('2d');

检查支持性

通过测试getContext方法的存在，可检查canvas的支持性

var canvas = document.getElementById('tutorial');
if(canvas.getContext){
    var ctx = canvas.getContext('2d');
    // code here
}else{
    // code here
}

canvas基础使用

栅格与坐标

  1. canvas元素默认被网格覆盖，通常来说网格的一个单元相当于canvas元素中的一像素
  2. 栅格起点为左上角(坐标为(0,0)),向右为X轴正方向，向下为Y轴正方向，所有元素的位置都是相对于起点
  3. 起点的坐标也会发生改变

路径

  1. 需要创建路径起始点
  2. 使用命令画出路径
  3. 可以闭合路径，渲染图形

函数

beginPath()

新建一条路径，生成之后，图形绘制命令被指向到路径上生成路径
本质上，路径是由许多子路径构成的，这些子路径都是在一个列表中，所有的子路径（线、弧形等）构成图形
每次调用该方法后，列表清空重置，然后可以重新绘制新的图形

moveTo(x,y)

将笔触移动到指定的坐标上
该方法并不能画出任何东西，只是移动“作画”的开始点

lineTo(x,y)

绘制一条从当前位置到指定坐标位置的直线
这里的“当前位置”可以通过moveTo()来改变

closePath()

闭合路径之后图形绘制命令又重新指向到上下文中
该方法会通过绘制一条从当前点到开始点的直线来闭合图形，因此如果图形已经闭合或使用fill()填充，该方法
就不是必需的，但对于stroke()，如果当前点不是开始点，可以调用该方法来闭合图形

stroke()

通过线条来绘制图形轮廓

fill()

通过填充路径内容来生成实心图形

clip()

创建新的裁切路径，默认情况下canvas有一个和它自身一样大的裁切路径（即没有裁切效果）

裁切路径属于canvas状态的一部分，可以使用save()保存

裁切路径创建之后所在出现在它里面的东西才会画出来

示例：

ctx.beginPath();
ctx.moveTo(75,50);
ctx.lineTo(100,75);
ctx.lineTo(100,25);
//  ctx.lineTo(75,50); // 对于fill，此行可省略；对于stroke，此行和closePath()选一才会使图形闭合，
否则只会生成两条线段
//  ctx.closePath();
ctx.fill();
//  ctx.stroke();

function draw() {
var ctx = document.getElementById('canvas-01').getContext('2d');
ctx.fillRect(0,0,150,150);
ctx.translate(75,75);

// Create a circular clipping path
ctx.beginPath();
ctx.arc(0,0,40,0,Math.PI*2,true);
ctx.clip();

// draw background
var lingrad = ctx.createLinearGradient(0,-75,0,75);
lingrad.addColorStop(0, '#232256');
lingrad.addColorStop(1, '#143778');
ctx.fillStyle = lingrad;
ctx.fillRect(-75,-75,150,150);

// draw stars
for (var j=1;j<50;j++){
    ctx.save();
    ctx.fillStyle = '#fff';
    ctx.translate(75-Math.floor(Math.random()*150),75-Math.floor(Math.random()*150));
    drawStar(ctx,Math.floor(Math.random()*4)+2);
    ctx.restore();
}

}
function drawStar(ctx,r){
    ctx.save();
    ctx.beginPath()
    ctx.moveTo(r,0);
    for (var i=0;i<9;i++){
    ctx.rotate(Math.PI/5);
    if(i%2 == 0) {
        ctx.lineTo((r/0.525731)*0.200811,0);
    } else {
        ctx.lineTo(r,0);
    }
    }
    ctx.closePath();
    ctx.fill();
    ctx.restore();
}

弧形

函数

arc(x, y, radius, startAngle, endAngle, anticlockwise)

绘制以(x,y)为圆心，以radius为半径，起始角度为startAngle，结束角度为endAngle，绘制方向为 anticlockwise 的圆弧

其中，startAngle/endAngle 以X轴为基准，单位为弧度，转化关系为：radians = (Math.PI / 180) * degrees

anticlockwise为布尔值，为true时是逆时针方向

示例

ctx.arc(100,100,50,Math.PI / 2, Math.PI,true);
ctx.stroke();

ctx.beginPath();
ctx.arc(100,100,50,- Math.PI / 2, Math.PI,true);
ctx.stroke();

ctx.beginPath();
ctx.arc(100,200,50,Math.PI / 2, Math.PI,false);
ctx.stroke();

ctx.beginPath();
ctx.arc(250,200,50,- Math.PI / 2, Math.PI,false);
ctx.stroke();

canvas样式

颜色

图形的填充颜色

fillStyle = color;

图形的轮廓颜色

strokeStyle = color;

color 可以是CSS颜色值的字符串， 渐变对象或者图案对象。默认情况下，线条和填充色都是黑色
(#000000)

对于颜色值，应该符合CSS3颜色值标准

ctx.fillStyle = "orange";
ctx.fillStyle = "#FFA500";
ctx.fillStyle = "rgb(255,165,0)";
ctx.fillStyle = "rgba(255,165,0,1)";

透明度 (transparency)

globalAlpha = transparencyValue

该属性影响canvas中所有图形的透明度，范围在0.0(完全透明)到1.0(完全不透明)，默认是1.0
该属性在绘制大量拥有相同透明度的图形时比较高效，但使用rgba()的方法定义颜色操作性更强

示例：

 ctx.fillStyle = '#FD0';
 ctx.fillRect(0,0,75,75);
 ctx.fillStyle = '#6C0';
 ctx.fillRect(75,0,75,75);
 ctx.fillStyle = '#09F';
 ctx.fillRect(0,75,75,75);
 ctx.fillStyle = '#F30';
 ctx.fillRect(75,75,75,75);
 ctx.fillStyle = '#FFF';

 // 设置透明度值
 ctx.globalAlpha = 0.2;

 // 画半透明圆
 for (var i=0;i<7;i++){
     ctx.beginPath();
     ctx.arc(75,75,10+10*i,0,Math.PI*2,true);
     ctx.fill();
 }

线型 (Line styles)

lineWidth

设置线条宽度。必须为正数，默认为1.0

示例：

ctx.moveTo(10,10);
ctx.lineTo(10,20);
ctx.moveTo(13.5,10);
ctx.lineTo(13.5,20);
ctx.stroke();

以上两条线虽然均为1px宽，但显示效果却不太相同

以宽度为1px的线条为例，该线条会在路径两边各延伸半像素。对于(13.5,10)到(13.5,20)的线条，由于延伸的半
像素，线条边缘正好落在像素边界，填充出来的就是准确的宽为1px的线条

因此线条宽度的奇偶性会影响真实渲染出的线条效果

lineCap

设置线条末端样式

lineCap = butt/round/square; // 默认值为butt

示例：

// 创建路径
ctx.strokeStyle = '#09f';
ctx.beginPath();
ctx.moveTo(10,10);
ctx.lineTo(140,10);
ctx.moveTo(10,140);
ctx.lineTo(140,140);
ctx.stroke();

// 画线条
ctx.strokeStyle = 'black';
for (var i=0;i<lineCap.length;i++){
    ctx.lineWidth = 15;
    ctx.lineCap = lineCap[i];
    ctx.beginPath();
    ctx.moveTo(25+i*50,10);
    ctx.lineTo(25+i*50,140);
    ctx.stroke();
}

lineJoin

设置线条接合处的样式

取值

  1. miter (默认)
    线段会在连接处外侧延伸直至交于一点，延伸效果受到miterLimit属性的制约
  2. bevel
  3. round

miterLimit

限制当两条线相交时交接处最大长度；所谓交接处长度（斜接长度）是指线条交接处内角顶点到外角顶点的长度

getLineDash()

返回一个包含当前虚线样式，长度为非负偶数的数组

setLineDash(segments)

设置当前虚线样式

lineDashOffset

设置虚线样式的起始偏移量

示例：

var offset = 0;

function draw() {
    ctx.clearRect(0,0, canvas.width, canvas.height);
    ctx.setLineDash([10, 2]);
    ctx.lineDashOffset = -offset;
    ctx.strokeRect(10,10, 100, 100);
}

function march() {
    offset++;
    if (offset > 16) {
    offset = 0;
    }
    draw();
    setTimeout(march, 20);
}

march();

createRadialGradient(x1, y1, r1, x2, y2, r2)

  • image可以是一个Image对象的引用，或者另一个canvas对象

阴影 (Shadows)

shadowOffsetX/shadowOffsetY = float

设定阴影在 X 和 Y 轴的延伸距离，它们是不受变换矩阵所影响的。负值表示阴影会往上或左延伸，正值则表示会
往下或右延伸，它们默认都为 0

shadowBlur = float

设定阴影的模糊程度，其数值并不跟像素数量挂钩，也不受变换矩阵的影响，默认为 0。

shadowColor = color

标准的 CSS 颜色值，用于设定阴影颜色效果，默认是全透明的黑色

示例：

ctx.shadowOffsetX = 2;
ctx.shadowOffsetY = 2;
ctx.shadowBlur = 2;
ctx.shadowColor = "rgba(0, 0, 0, 0.5)";

ctx.font = "20px Times New Roman";
ctx.fillStyle = "Black";
ctx.fillText("Sample String", 5, 30);

两个可能的值：

  • evenodd

canvas状态的保存和恢复(Saving and Restoring state)

save() 与 restore()

save() 方法用来保存当前状态， restore() 用来恢复前一个保存的状态

canvas状态是以堆(stack)的方式保存的，每一次调用save()方法，当前状态就会被推入堆中保存起来，状态
包括：

  • 当前应用的变形
  • strokeStyle/fillStyle/globalAlpha/lineWidth/lineCap/lineJoin/miterLimit/shadowOffsetX/shadowOffsetY/shadowBlur/shadowColor/globalCompositeOperation
    的值
  • 当前的裁切路径(clipping path)

每使用一次restore()方法，上一次保存的状态就从堆中弹出，所有设定恢复

示例：

ctx.fillRect(0,0,150,150);   // Draw a rectangle with default settings
ctx.save();                  // Save the default state

ctx.fillStyle = '#09F'       // Make changes to the settings
ctx.fillRect(15,15,120,120); // Draw a rectangle with new settings
ctx.save();                  // Save the current state

ctx.fillStyle = '#FFF'       // Make changes to the settings
ctx.globalAlpha = 0.5;
ctx.fillRect(30,30,90,90);   // Draw a rectangle with new settings

ctx.restore();               // Restore previous state
ctx.fillRect(45,45,60,60);   // Draw a rectangle with restored settings

ctx.restore();               // Restore original state
ctx.fillRect(60,60,30,30);   // Draw a rectangle with restored settings

canvas变形

变形前先保存状态是一个好习惯，调用restore()可以快速恢复到原先的状态

移动(translate)

translate(x, y)

移动canvas，使其原点从(0,0)移动至(x, y)

旋转(rotate)

rotate(angle)

以canvas的原点为中心，顺时针旋转一个角度，角度以弧度为单位

绽放(scale)

scale(x, y)

x/y 分别代表X/Y轴的缩放因子，均为正值。小于1时缩小，大于1时放大

变形(transform)

transform(m11,m12,m21,m22,dy,dy)

该方法必须将当前的变形矩阵与下面的矩阵相乘

m11 m21 dx
m12 m22 dy
0   0   1

setTransform(m11,m12,m21,m22,dx,dy)

该方法必须重围当前的变形矩阵为单位矩阵，然后以相同的参数调用transform方法

示例：

var sin = Math.sin(Math.PI/6);
var cos = Math.cos(Math.PI/6);
ctx.translate(200, 200);
var c = 0;
for (var i=0; i <= 12; i++) {
    c = Math.floor(255 / 12 * i);
    ctx.fillStyle = "rgb(" + c + "," + c + "," + c + ")";
    ctx.fillRect(0, 0, 100, 10);
    ctx.transform(cos, sin, -sin, cos, 0, 0);
}

ctx.setTransform(-1, 0, 0, 1, 200, 200);

canvas组合(composite)

处理图形之间的显示与遮盖问题

globalCompositeOperation = type

示例：

ctx.fillStyle = 'blue';
ctx.fillRect(0,0,100,100);

ctx.globalCompositeOperation = type;

ctx.fillStyle = 'red';
ctx.arc(100,100,50,0,Math.PI*2);
ctx.fill();
type 说明 示例
source-over 默认，新图形覆盖在原图形上
source-in 新图形重叠部分出现，其余部分透明
source-out 原图形透明，新图形只出现与原图形不重叠的部分
source-atop 新图形透明，重叠部分出现，原图形保留
lighter 图形重叠部分作加色处理
xor 重叠部分透明
destination-over 新图形出现在原图形下面
destination-in 原图形重叠部分出现，其余均透明
destination-out 新图形透明，原图形只出现与新图形不重叠的部分
destination-atop 原图形重叠部分保留，原图形其余部分透明，新图形在原有内容下绘制
darker 重叠部分作减色处理
copy 新图形保留，其余均清除
